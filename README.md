#### 프로젝트 소개
1. 가상 환경
    ###### venv
        ### for mac
        $ python3 -m venv druid-batch
        
        ### for linux
        $ python3.7 -m venv druid-batch
        
        $ . druid-batch/bin/activate
2. requirements
    ###### 설치
        $ pip install -r pip-requirements
       
3. 실행
    ###### pyhton 실행
        $ python3 main.py --target GENIEWORKS LANDINGTYPE CAMPAIGNDATA TMSDATA TMSDATA2 TMSDATA3
